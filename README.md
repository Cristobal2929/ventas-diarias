# Registro de ventas diarias

Esta aplicación permite a un negocio registrar cada venta del día, ver la lista completa de ventas y el total acumulado.

## Características

- Formulario sencillo para añadir una venta (descripción y monto).
- Tabla dinámica que muestra todas las ventas registradas.
- Campo que muestra el total de ventas en tiempo real.

## Uso

1. Introduce la descripción de la venta y el monto.
2. Pulsa **Agregar venta**.
3. La tabla se actualizará con la nueva entrada y el total se recalculará automáticamente.

## Instalación

El Space está listo para ejecutarse en Hugging Face. No se requieren configuraciones adicionales.
